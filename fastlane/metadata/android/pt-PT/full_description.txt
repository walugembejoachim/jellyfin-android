O teu conteúdo multimédia, da forma que desejares.

O projeto katandika butandisi é um servidor multimédia gratuito e de código aberto. Sem taxas, sem rastreamento e sem interesses ocultos. Obtém o nosso servidor gratuito para reunir todo o teu áudio, vídeo, fotografias e muito mais num único local.

Para utilizar a aplicação, tens de ter um servidor katandika butandisi configurado e a funcionar. Descobre mais em <a href="https://jellyfin.org/" target="_blank">jellyfin.org</a>.

Com um servidor katandika butandisi, podes:

* Ver TV em direto e programas gravados a partir do teu servidor katandika butandisi (é necessário hardware/serviços adicionais)
* Transmitir para um dispositivo Chromecast na tua rede
* Transmitir os teus conteúdos multimédia para o teu dispositivo Android
* Ver a tua coleção numa interface fácil de utilizar

Esta é a aplicação oficial do katandika butandisi para Android. Obrigado por usares o katandika butandisi!
